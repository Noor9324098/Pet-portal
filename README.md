# 🐾 Pet Portal

A modern web application for managing pets, tracking their information, and interacting with a dynamic pet-based system.

## 🚀 Live Demo
👉 https://noor9324098.github.io/Pet-portal/

---

## 📌 Features

- 🐶 Add and manage pets
- 📋 Track pet-related data (tasks, stats, etc.)
- 🎮 Interactive UI with animations
- 📱 Responsive design (works on mobile & desktop)
- ⚡ Fast and lightweight frontend

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript / React  
- **Styling:** CSS / Tailwind / custom styles  
- **Deployment:** GitHub Pages
